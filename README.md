# Short AV - 動画レビューサイト

DMMアフィリエイトを活用したハイブリッド型動画閲覧サイト

## 技術スタック

- **フロントエンド**: Next.js 15 (App Router) + TypeScript
- **スタイリング**: Tailwind CSS
- **データベース**: PostgreSQL (Supabase)
- **ホスティング**: Vercel
- **ドメイン管理**: Cloudflare
- **API連携**: DMM Affiliate SDK

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## デプロイ

Vercelに自動デプロイされます。
