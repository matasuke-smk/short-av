## アフィリエイトID復元手順

新しいアフィリエイトIDが発行されたら、以下の手順で復元してください。

### 📝 詳細手順
詳細は `config/affiliate.md` を参照してください。

### 🚀 クイックガイド

1. **VideoSwiper.txを編集** (264行目付近)
```tsx
// 変更前
href="...&ch_id=1082_640_200"

// 変更後
href="...&ch_id=1082_640_200&af_id=新しいID"
```

2. **AgeVerificationGate.tsx を編集** (77行目付近)
```tsx
// 変更前
href="...&ch_id=1082_300_250"

// 変更後
href="...&ch_id=1082_300_250&af_id=新しいID"
```

3. **コミット＆デプロイ**
```bash
git add app/components/VideoSwiper.tsx app/components/AgeVerificationGate.tsx
git commit -m "アフィリエイトIDを復元: 新しいID"
git push
```

### 🔍 検索方法
- ファイル内検索: `af_id`、`ch_id=1082`
- 設定箇所: 2箇所（VideoSwiper.tsx、AgeVerificationGate.tsx）
- 元のID: `matasuke-002`（参考用）
