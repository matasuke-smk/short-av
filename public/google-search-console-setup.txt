# Google Search Console 所有権確認ファイル配置場所

このディレクトリ（public/）に、Google Search Consoleから取得した
確認用HTMLファイルを配置してください。

## 手順

1. Google Search Console (https://search.google.com/search-console/) にアクセス
2. プロパティを追加（https://short-av.com）
3. 「HTMLファイル」による確認を選択
4. ダウンロードしたファイル（例: google1234567890abcdef.html）を
   このディレクトリ（public/）に配置
5. git add, commit, push してデプロイ
6. GSCで「確認」ボタンをクリック

## ファイル名の例

google1234567890abcdef.html
google9876543210fedcba.html

## 確認後

確認が完了した後もファイルは削除しないでください。
Googleは定期的に再確認を行います。

詳細は docs/Google-Search-Console-連携ガイド.md を参照してください。
