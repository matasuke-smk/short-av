## 実装可能な改善施策

### 1. 🚀 即効性のある施策（1-2週間で実装可能）

#### A. PWA（Progressive Web App）対応
**効果**: スマホのホーム画面に追加可能にし、アプリのような体験を提供
```javascript
// manifest.jsonを追加
{
  "name": "Short AV",
  "short_name": "ShortAV",
  "start_url": "/",
  "display": "standalone",
  "theme_color": "#000000",
  "background_color": "#000000",
  "icons": [...]
}

// Service Workerでオフライン対応
// キャッシュによる高速化
```

#### B. プッシュ通知機能
**効果**: リピート率30-50%向上
- 新着動画の通知
- お気に入り女優の新作通知
- 週間ランキング更新通知

#### C. シェア機能の強化
**効果**: 自然な拡散による新規ユーザー獲得
```javascript
// 各動画にシェアボタンを追加
const shareVideo = async () => {
  await navigator.share({
    title: video.title,
    text: 'この動画おすすめ！',
    url: `https://short-av.com?v=${video.id}`
  });
};
```

#### D. ダークモード/ライトモード切り替え
**効果**: ユーザー満足度向上、滞在時間増加
- システム設定連動

- 設定の永続化
